# 🛡️ Network Security Toolkit

A lightweight Python-based network security toolkit for educational purposes and authorized security assessments. This project provides useful utilities for gathering information about a network and generating basic security reports.

«Disclaimer: This tool is intended only for use on networks and systems that you own or have explicit permission to assess. The author is not responsible for any misuse.»

---

# ✨ Features

- 🌐 Network Host Discovery
- 📡 Local Network Information
- 🖥️ Hostname Detection
- 🔍 TCP Port Scanner (for authorized systems)
- 📊 Colored Terminal Interface
- 📁 Report Generation
- ⚡ Fast Multi-threaded Scanning
- 📝 Logging Support

# 📦 Installation

Clone the repository:
```bash
https://github.com/skvirus711/wifi-vernability-scanner.git
cd wifi-vernability-scanner
```
Install dependencies:
```bash
pip install -r requirements.txt
```
---

# 🚀 Usage

Run the toolkit:
```bash
python main.py
```
Example menu:

====================================
      NETWORK SECURITY TOOLKIT

[1] Network Scan

# 📋 Requirements

Install manually:
